# wazuh-siem-detection-lab
A hands-on home lab environment built to simulate real world SOC monitoring using Wazuh SIEM.

## 🎯 Objective
To gain practical experience in:
- Log analysis
- Threat detection
- Incident investigation

---

## 🛠️ Lab Setup
- Ubuntu VM for Wazuh Manager
- Windows VM to Monitored endpoint
- Kali Linux for Attack simulation

---

## ⚙️ Implementation

### 🔹 Log Collection
- Configured Wazuh agents
- Collected authentication and system logs

---

### 🔹 Detection Rules
- Created rules for:
  - Failed login attempts
  - Suspicious authentication activity

---

### 🔹 Attack Simulation
- Simulated brute-force login attempts
- Observed alerts in Wazuh dashboard
